如何搭建一个博客

何利用hexo框架搭构一个博客

内容
1：首先安装node.js,下载地址为：https://nodejs.org/en/
2：CMDj进入dos命令窗口 node -v可查看版本，npm -v即包的版本
3：镜像源，由于国内慢,所以用淘宝的,命令为npm install -g cnpm –registry=http://registry.npm.taobao.org
4: cnpm 管理器,自动补全，说明工具有了
5: 查看cnpm版本 cnpm -v
6: cnpm install -g hexo-cli #安装hexo框架cnpm install -g hexo-cli #安装hexo框架
7:hexo -v #查看hexo版本
8:mkdir blog #创建blog目录
9:cd blog #进入blog目录
10:sudo hexo init #生成博客 初始化博客
11:hexo s #启动本地博客服务
12:http://localhost:4000/ #本地访问地址
13:hexo n “我的第一篇文章” #创建新的文章
#返回blog目录
hexo clean #清理
hexo g #生成
#Github创建一个新的仓库 YourGithubName.github.io
cnpm install –save hexo-deployer-git #在blog目录下安装git部署插件
#配置_config.yml
# Deployment
## Docs: https://hexo.io/docs/deployment.html
deploy:
type: git
repo: https://github.com/YourGithubName/YourGithubName.github.io.git
branch: master
hexo d #部署到Github仓库里
https://YourGithubName.github.io/ #访问这个地址可以查看博客

git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia #下载yilia主题到本地

#修改hexo根目录下的 _config.yml 文件 ： theme: yilia

hexo c #清理一下
hexo g #生成
hexo d #部署到远程Github仓库
https://YourGithubName.github.io/ #查看博客
